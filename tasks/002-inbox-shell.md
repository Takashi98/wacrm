# Task 002 - Inbox Page Shell

## Goal
Build the Inbox page shell for WACRM.

## Scope
This task includes:
- create a routed inbox page
- create a desktop layout with:
  - left conversation list
  - center chat panel
  - right lead info panel
- create a responsive mobile layout
- use only mock data
- include empty states where useful

## Must include
- route: /inbox
- page title/header
- conversation list UI
- chat message area UI
- lead/contact sidebar UI
- clean responsive layout for both mobile and desktop

## Out of scope
- real-time chat
- WhatsApp integration
- backend API integration
- lead editing logic
- assignment logic
- sending real messages

## Important rules
- keep it UI shell only
- no product logic yet
- use reusable components
- keep design simple and premium
- make layout easy to extend later

## Done when
- /inbox route works
- desktop shell is visible
- mobile layout works
- mock data is visible
- no backend integration is required yet
