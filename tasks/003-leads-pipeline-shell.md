# Task 003 - Leads Pipeline Shell

## Goal
Build the Leads page shell for WACRM.

## Scope
This task includes:
- create a routed leads page
- create a pipeline board UI
- show columns for:
  - New
  - Contacted
  - Qualified
  - Won
  - Lost
- use mock lead cards only
- include desktop and mobile responsive layouts
- include empty states where useful

## Must include
- route: /leads
- page header
- pipeline columns
- mock lead cards
- card details like:
  - lead name
  - business name
  - source
  - owner
  - estimated value
  - tag or stage note

## Out of scope
- drag and drop
- backend integration
- editing logic
- filters logic
- search logic
- real pipeline actions

## Important rules
- keep it UI shell only
- no product logic yet
- keep design aligned with inbox shell
- make it feel like a premium SaaS CRM
- support both desktop and mobile

## Done when
- /leads route works
- pipeline shell is visible
- mock data is shown
- mobile layout works
- no backend integration is required
